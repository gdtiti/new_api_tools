## 实际改动摘要

- 本轮改动:
  - 新增渠道健康度与并发估算后端服务/路由，并把健康得分、健康状态、估算 peak/p95 并发接到 API。
  - 新增订阅统计后端服务/路由，运行时探测 `subscription_plans` / `user_subscriptions` schema，缺表或缺关键列时显式返回 unsupported。

  - 把 `ChannelInsights`、`SubscriptionAnalytics`、`TemporaryAccounts` 接入 `App.tsx`、`Layout.tsx`、`components/index.ts`。
